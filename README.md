**Task Manager with Nuxt 3**

Stack used:
 - Vue 3
 - Nuxt 3
 - Vue Draggable
 - SortableJS
 - TypeScript
 - Tailwindcss

Implemented features:
  - Drag & Drop
  - Task Deletion
  - Task Creation
  - Column Deletion
  - Column Creation
  - Column Editing
  - Data Saved via Local Storage

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```
